## 待办：上下文与技能

### 滚动摘要阻塞耗时

- **状态**：待优化  
- **说明**：实际逻辑在 `session/buildModelContext.ts`（调用 `summarizeMessages`），非 `createServer.ts`。当历史被折叠时，用户需等待摘要 LLM 调用 + 本轮生成，**串行耗时**仍在。  
- **可选方向**：异步摘要、降低摘要频率、或接受精度/延迟权衡。

### Skills 仅支持 JSON

- **状态**：待增强  
- **说明**：`loadSkills.ts` 仅加载 `.json`；内容不符合 Skill 结构时会解析失败。  
- **可选方向**：支持 `.md`  frontmatter、或更友好的校验报错。
