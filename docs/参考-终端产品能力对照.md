# 终端产品能力对照（OpenClaw / Claude Code → OneClaw）

> 信息来自公开文档与社区文章，**不保证与闭源版本逐条一致**，仅作产品对标与排期参考。  
> 更新：2026-04-09

---

## 1. OpenClaw（TUI + Gateway）

| 能力 | 说明 | OneClaw 现状 |
|------|------|--------------|
| TUI 连 Gateway | 默认 `ws://127.0.0.1:18789`，可 `--url` / 鉴权 | 有本机 WS + Ink TUI，端口默认 `18789`，可 `-p` / `ONECLAW_TUI_WS_PORT` |
| 顶栏信息 | URL、agent、session | 有：WS、session；`--agent` / `--task` 可传 |
| 聊天区 | 用户 / 助手 / 系统 / 工具卡片 | 有用户 / 助手 / 错误行；**无工具卡片、无流式逐字** |
| 状态行 | connecting / streaming / idle / error 等 | 有 connected / 等待回复；**无 streaming 细分** |
| 底栏 | agent、session、model、token、thinking 等 | 有 model 名（ready 包）、**无 token 计数与 thinking** |
| 历史加载 | 连接后可拉最近 N 条（如 200） | **无**，仅本会话内存列表 |
| 流式输出 | 流式更新至定稿 | **无**（整段回复后一次推送） |
| 快捷键 | Esc 中止、Ctrl+L 模型、Ctrl+G agent、Ctrl+P session 等 | **仅 Ctrl+C 退出** |
| 多行输入 / 编辑器 | 增强输入 | **单行** `ink-text-input` |
| 图片 / 富媒体 | iTerm2 MEDIA 等 | **无** |
| 会话范围 | per-sender / global 等 | 等价于 `sessionKey` + 可选 `taskId` |

参考：OpenClaw 文档中 TUI / Gateway 章节（各镜像站标题多为「TUI」「gateway」）。

## 2. Claude Code（CLI + 斜杠命令）

| 能力 | 说明 | OneClaw 现状 |
|------|------|--------------|
| 单命令启动 | 安装后终端输入 `claude` | 可用 **`pnpm link --global` 后 `oneclaw`**；本仓库 `pnpm cli` 无参时也会**默认进入 TUI** |
| 斜杠命令 | `/help`、`/clear`、`/compact`、`/resume`、`/model`、`/doctor`、`/context`、`/cost` 等 | REPL 有 `/help`、`/session`、`/clear`、`/status`；**无 compact、resume、cost、内置 /model 切换** |
| 自定义斜杠 | 如 `.claude/commands/` 下 Markdown | **无** |
| 会话与压缩 | `/compact`、上下文统计 | **无**；摘要逻辑在服务端对话链里，**无 TUI 专用 /context** |
| 工程初始化 | `/intro`、`CLAUDE.md` | **无**；可有 workspace `agents.json` / skills |
| 非交互模式 | 管道、单次问答、`--output-format` 等 | 部分靠 `curl` / API；**CLI 未统一做 pipe 模式** |
| 模型切换 | `/model`、CLI `--model` | **环境变量 / 配置**；**无运行中切换** |
| 诊断 | `/doctor` | 有子命令 **`oneclaw doctor`** |

参考：Anthropic Claude Code 文档「CLI usage」、社区整理的 Slash Commands 列表。

## 3. OneClaw 建议演进顺序（若要对齐体验）

1. **TUI**：流式 delta（WS 协议 + `handleUnifiedChat` 改造或边侧回调）、底栏 token 估算、Esc 取消当前请求。  
2. **会话**：连接时拉转录最近 N 条（需服务端按 session 暴露只读接口或 WS 首包）。  
3. **快捷键**：模型 / session 选择（可先简化为 `?` 弹出说明）。  
4. **斜杠**：在 TUI/REPL 内统一解析 `/`，逐步加 `/compact` 调用已有摘要链路等。  
5. **全局体验**：发布 npm 包或安装器，使 `oneclaw` 不依赖 `pnpm link`（当前以 link 为主）。
